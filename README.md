# Library App (Diploma Project)

Приложение для автоматизации работы библиотеки. Реализовано как десктопное приложение на базе Electron, React, Node.js и PostgreSQL.

## Технологии

- Electron — оболочка десктопного приложения  
- React — пользовательский интерфейс  
- Node.js + Express — backend сервер  
- PostgreSQL — база данных  

## Основные возможности (план)

- Авторизация читателей и сотрудников  
- Просмотр каталога книг и информации об издательствах  
- Бронирование книг читателями  
- Одобрение или отказ заявок сотрудником с указанием причины  
- Учёт выданных и возвращённых книг  

## Структура проекта

- `src/main` — главный процесс Electron (создание окна приложения)  
- `src/renderer` — React-интерфейс (frontend)  
- `src/server` — Express-сервер и API  
- `src/database` — скрипты и конфигурация PostgreSQL  
- `public` — статические файлы и HTML-шаблон  

## Запуск проекта

### Требования

- Node.js (LTS)  
- PostgreSQL 16+  

### Шаги

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/library-app.git
   cd library-app
   ```

2. Установить зависимости:
   ```bash
   npm install
   ```

3. Создать файл `.env` в корне проекта и указать настройки базы данных:
   ```env
   DB_HOST=localhost
   DB_PORT=5432
   DB_USER=postgres
   DB_PASSWORD=ВАШ_ПАРОЛЬ
   DB_NAME=library_db
   SERVER_PORT=5000
   ```

4. Создать базу данных `library_db` и выполнить скрипт `src/database/init.sql`.
   4.1 Выполни в `psql` команду:
```sql
CREATE DATABASE library_db;
```
   4.2 Подключение к базе
```sql
\c librtary_bd
```
4.3 Выполнить скрипт
```sql
\i 'C:/путь_к приложению/library-app/src/database/init.sql'
```
⚠️ Важно: Замени путь на свой (где у тебя лежит проект), используй прямые слэши /

6. Запустить приложение в режиме разработки:
   ```bash
   npm run dev
   ```

После запуска откроется окно Electron с интерфейсом приложения.
